Парсит сайт kinogo.club после запуска подсчитывается количество страниц для каждого жанра, что бы вывести информацию. Полсе этого следует ввести цифру которая соответствует жанру и затем ввести количество страниц которое нужно спарсить, после этого создаться файл parser.xml который следует открыть в Excele. К сожалению парсер работает достаточно медленно.
